Base de données des alumni

Contexte

Projet tutoré réalisé dans un cadre universitaire visant à concevoir une base de données pour la gestion des anciens étudiants.

Rôle

Conception de la structure de la base, implémentation SQL et gestion des données.

Travaux réalisés

  • Modélisation de la base de données
  • Création de tables et relations SQL
  • Insertion et sécurisation des données
  • Documentation technique

Outils utilisés

  • SQL / MySQL
  • Outils de modélisation
